# Phase Context -- Define

This is the Define phase -- convergent synthesis of the Discover findings. The goal is to move from a wide set of observations and patterns to a focused, evidence-backed problem definition.

Synthesize Discover findings into problem statements, needs-based personas, journey maps, and opportunity areas. Prioritize and frame the opportunities based on citizen impact, frequency, and emotional weight. Every problem statement must be traceable to evidence from the Discover phase.

Output should give the municipality and the KOOS team a clear, shared understanding of where the biggest opportunities lie -- framed from the citizen perspective, not the organizational perspective. The Define phase output guides all solution development in the next phase.
